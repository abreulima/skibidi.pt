O que é Frontend Development?

É o desenvolvendo da interface gráfica de um site, as quais os usuários têm acesso. A composição atual do desenvolvimento frontend é feito através de HTML, CSS e JavaScript.

Resumidamente, HTML é conteúdo do site. CSS é o aspeto, e JavaScript é o comportamento.

Mas React, Angular, Vuejs...etc?

Essas ferramentas baseam-se em HTML, CSS e JavaScript e são conhecidas como frameworks. A framework dita como a estrutura de uma aplicação deve ser desenvolvida.

Como a Web Funciona?

A internet é composta por uma relação cliente-servidor. O cliente, através do navegador, envia e recebe informações para o servidor, o qual nos devolve HTML, CSS, JavaScript, imagens, vídeos entre outras coisas, e por fim, o navegador os exibe no ecrã.

Como desenvolver frontend, nosso trabalho é focado apenas no navegador. Como as requisões HTTP funcionam, e como o servidor trata de enviar os dados para o cliente o qual pediu é trabalho do desenvolver backend.

HTML

HyperText Markup Language é a linguagem de marcação (não programação) que define a estrutura de um site.

Uma página HTML é composto por elementos HTML, imagens, videos, JavaScript, CSS...
